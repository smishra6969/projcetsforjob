# Cloud Resume Website on AWS

This project is a personal resume website hosted on AWS using modern cloud services.

## 🚀 Technologies Used
- HTML, CSS, JavaScript
- Amazon S3 (Static Website Hosting)
- Amazon Route 53 (Domain & DNS)
- Amazon CloudFront (CDN & HTTPS)
- AWS Certificate Manager (SSL/TLS)

## 🌐 Live Website
getjobbynewproject.com

## 📌 Features
- Fully responsive resume website
- Hosted on AWS S3
- Custom domain using Route 53
- Secure HTTPS using CloudFront & SSL certificate
